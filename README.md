## Features

- Syntax highlighting for SBML & SBSS
- Auto-completion
   - well-known property values
- Diagnostics
   - Error for invalid value for well-known property
   - Error for dangling `=end` in *.sbml
   - Warning `=begin` / `=end` tag mismatch in *.sbml
